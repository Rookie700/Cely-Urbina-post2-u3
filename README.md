# Cely-Urbina-post2-u3
Este laboratorio se enfoca en el ensamblado directo de programas en memoria utilizando el comando A del depurador DEBUG en el entorno DOSBox
. El objetivo es realizar una ejecución controlada paso a paso mediante el comando T para observar cómo cambian los registros y las banderas (FLAGS) del procesador 8086
. Además, se analiza el funcionamiento de estructuras de control iterativas mediante el uso del registro CX y la instrucción LOOP, finalizando con un desglose técnico de la codificación en código máquina
.
